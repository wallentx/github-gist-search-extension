# Privacy Policy

Last updated: March 11, 2026

GitHub Gist Enhanced Search is a Chrome extension that improves search on `gist.github.com` for logged-in GitHub users.

## What the extension does

The extension enhances the existing gist search experience by providing a GitHub-style query builder, qualifier suggestions, autocomplete, inline query highlighting, and helper shortcuts such as `@me`.

## Data the extension handles

To provide its single purpose, the extension may process the following data while you use `gist.github.com`:

- website content on `gist.github.com`, including the search input and related page content
- user activity related to the search experience, such as typing, clicks, focus, and keyboard navigation

This processing happens locally in your browser as part of the extension's user interface and search features.

## How data is used

This data is used only to provide the extension's search-related functionality on `gist.github.com`.

The extension does not use handled data for advertising, profiling, creditworthiness, lending decisions, or unrelated analytics.

## Data sharing and selling

GitHub Gist Enhanced Search does not sell user data.

GitHub Gist Enhanced Search does not transfer user data to third parties.

The extension does not "call home," does not send your search queries or page content to a developer-operated server, and does not use remote analytics or tracking services.

## Remote code and external services

The extension does not load remote code.

The extension does not require a separate account, and it does not send handled data to a dedicated backend operated for this extension.

## Data retention

Handled data is processed in the browser as part of the extension's functionality. The extension is not designed to maintain a separate remote database of user data.

## Scope

The extension is intended to operate on `gist.github.com`.
